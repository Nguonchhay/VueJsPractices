# vuex-app

## Project setup
```
yarn install
```

### Start local server
```
json-server data/todos.json -p 8085
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
